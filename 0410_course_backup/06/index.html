三种不同 Access 的方法

  • Playwright: 独立浏览器进程，完全程序化控制，适合自动化测试和无头爬取，但与用户当前浏览器完全隔离
  • Claude Chrome Extension: 注入用户真实浏览器，能读取已登录状态、Cookie 和当前页面上下文，但需要用户手动触发或保持扩展运行
  • CDP (Chrome DevTools Protocol) : 通过协议直连已运行的 Chrome 实例，可附加到用户现有 Tab 进行底层控制（网络拦截、JS 注入），需要 Chrome 以 `--remote-debugging-port` 启动

技巧: 纯想的三级上下文索引

  • 想象我们要让 Claude Code 快速地去找到商场里“外婆家员工休息室里张三的柜子”

  • “哪一个房间里的柜子” ? 如果知道外婆家里每个房间是干嘛的就好了: 厨房, 休息室...
  • “哪一个柜子?” 如果知道每个员工的柜子在哪儿就好了: 张三, 李四
