# EinClaw - Product website and web application
HTML + CSS + Vite + React 19 + TypeScript + TailwindCSS v4 + shadcn/ui

## DESIGN LAW

**All design must come from the design system.** No hardcoded hex/rgb/oklch values in components.
Every color is a CSS variable from design tokens. Every component is from `components/ui/`.

## Directory Structure

site/ - Static product website (HTML + CSS)
web/ - Frontend application (React 19 + TypeScript + Vite + TailwindCSS v4 + shadcn/ui, amethyst-haze theme)
bp/ - Standalone HTML business-plan and course collateral files
0410_course/ - Course slide exports, assets, and related course materials
vercel.json - Vercel deployment configuration
